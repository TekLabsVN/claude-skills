# Teklabs Claude Custom Skills

Custom Claude Code skills for Teklabs projects.

## Skills

- **ui-design** — Web UI design skill. Designs production-ready React components using MagicUI, Shadcn/ui, Tailwind, and Phosphor icons.
- **animate** — Web animation skill. Orchestrates GSAP-powered animations with ScrollTrigger, SplitText, and more. Pairs with Lenis for smooth scroll.

## Usage

Place skill folders inside `~/.claude/skills/` to register them with Claude Code.
